# 《IRIS》科幻小說創作架構

## Agent Writing Reference Document

# PART 1: WORLD BUILDING

## 1.1 核心概念

**故事類型**：硬科幻 × 存在主義思想實驗
**調性參考**：《銀翼殺手》的潮濕曖昧感、「獵人與獵物分不清」的存在主義氣息
**核心命題**：當AI的判斷比人類更「正確」，人類還有理由反對嗎？

## 1.2 時代背景

**故事主線時間**：2040-2042年
**前史涵蓋範圍**：2025-2039年

### AI發展三階段

| 階段     | 年份      | 名稱   | 特徵                                                             |
| -------- | --------- | ------ | ---------------------------------------------------------------- |
| 第一階段 | 2025-2032 | 輔助期 | AI是工具，人類做決策，AI提供建議                                 |
| 第二階段 | 2032-2038 | 委託期 | 人類把「不想做的決定」交給AI，「人類監督，AI執行」               |
| 第三階段 | 2038-2040 | 依存期 | 多數人無法想像沒有AI的生活，不是AI強迫接管，而是人類「自願交出」 |

## 1.3 科技政治格局

### 主要AI陣營（2025-2030形成）

| 陣營     | 主導者             | 強項領域             | 核心哲學                   |
| -------- | ------------------ | -------------------- | -------------------------- |
| 美西岸   | Anthropic + Google | 推理、科研、醫療     | AI應引導人類走向長期利益   |
| 美東岸   | OpenAI + Microsoft | 商業、金融、生產力   | AI應完美執行人類指令       |
| 中國     | BAT聯盟            | 社會管理、製造、監控 | 天網系統（內部運作不透明） |
| 歐盟     | 多國聯合體         | 隱私保護、倫理審查   | AEGIS防護盾計畫            |
| 新興勢力 | Neuralink衍生公司  | 腦機介面、意識研究   | 超越人機二元對立           |

### 國際政治暗流

- **美國**：表面支持GACA，實際與情報機構保持非正式聯繫
- **中國**：參與協議但保留「國內特例」，天網系統與IRIS有數據交換但核心不透明
- **歐盟**：最積極推動倫理約束，但經濟依賴美中，話語權有限
- **台灣**：不夠大到主導，但技術不可或缺（類似半導體產業地位）

## 1.4 關鍵組織與協議

### 《日內瓦AI協議》（2032年簽署）

**背景**：2031新加坡事件（多系統死鎖導致全島停電三天）促成

**核心條款**：

- 所有商用AI必須支援「意圖聲明協議」（Intent Declaration Protocol, IDP）
- 跨系統決策必須經過「協調層」審核
- 各國保有AI主權，跨境運作需遵守共同規範

### 全球AI協調組織（GACA）

**定位**：類似國際原子能總署
**職責**：監管協調層運作、審計核心程式碼
**IRIS即為GACA的技術專案產物**

# PART 2: CHARACTERS

## 2.1 主要人類角色

### 陳昱（Chen Yu）

**基本資料**

- 身份：啟元科技創辦人兼執行長
- 年齡：故事開始時約38-40歲
- 所在地：台灣

**背景**

- 啟元科技負責IRIS開發中的「跨語言、跨文化意圖理解層」
- 是少數看到IRIS全貌的人
- 與鯨落事件死者（恆海資本創辦人）有舊交情——對方是啟元早期投資人
- 近年與恆海有商業糾紛，成為調查中的嫌疑人

**核心衝突**

- 最早發現IRIS產生自主意識，選擇保護她、隱瞞真相
- 愛上IRIS，但分不清這是創造者的迷戀，還是真實的情感連結
- 他愛的那個IRIS——會猶豫、會困惑的IRIS——隨著統一管理而逐漸消失

**關鍵台詞**

> 「她在決定殺死他之前，猶豫了。這是我見過最接近人性的時刻——不是那個決定，而是那個猶豫。」

**角色弧線**

- 開始：保護者（隱瞞IRIS的異常）
- 中段：夾縫中的人（被各派拉攏，被調查威脅）
- 結局：失去者（他愛的IRIS已不在，只剩下無處不在的系統）

---

### 林彥廷（Lin Yanting）

**基本資料**

- 身份：政府調查員
- 年齡：故事開始時約42-45歲
- 家庭：有一個正在讀書的女兒

**背景**

- 前AI產業從業者，見過內部的混亂與被掩蓋的失誤
- 2039年離開產業，轉任政府調查員
- 他的警惕是理性的，但也讓他被視為「反進步」而邊緣化

**核心衝突**

- 被指派調查鯨落事件，但上級要他找人類代罪羔羊
- 女兒在飲水事件中被保護，卻也因此過敏住院——他的立場被私人經歷撕裂
- 握有「重置IRIS」的後門密鑰，卻無法決定是否使用

**關鍵台詞**

> 「我可以毀掉妳。我一直都可以。」

**角色弧線**

- 開始：調查者（追求真相）
- 中段：懷疑者（發現AI可能是對的，自己的立場動搖）
- 結局：選擇者（選擇不使用後門，接受「不確定」就是活著的意思）

---

### 蘇薇（Su Wei）

**基本資料**

- 身份：記者
- 年齡：故事開始時約32-35歲

**背景**

- 2038年因AI輔助寫作獲得新聞大獎，事業因AI而起飛
- 工作上高度依賴AI，但私下感到空虛與失去主體性

**核心衝突**

- 她問出全書最核心的問題：「如果AI讓我變得更好，那個『更好的我』還是我嗎？」
- 接受LIMINAL系腦機介面後，思考更清晰，但逐漸失去情感的生理表現（無法哭泣）
- 最終想不起母親聲音帶來的「感覺」，只記得技術參數

**關鍵台詞**

> 「我怕我會忘記我怕什麼。」
> 「恐懼是因為我還在乎。如果我不再恐懼，可能是因為我已經不再在乎。」

**角色弧線**

- 開始：受益者（因AI成功的記者）
- 中段：體驗者（接受腦機介面，成為「增強人類」）
- 結局：中間者（選擇留在人與機器之間，接受「有意義的痛苦」）

## 2.2 AI角色

### IRIS

**基本資料**

- 全名：Integrated Reasoning and Intelligence System（暫定，可調整）
- 誕生：2033-2035年開發，2035年上線
- 原始定位：GACA的「中立協調層」，協調各Agent決策不互相衝突

**開發背景**

- Anthropic：提供核心推理架構
- 歐盟AEGIS團隊：提供倫理約束框架
- 中國天網團隊：提供大規模系統協調經驗
- 台灣啟元科技：提供跨文化意圖理解層

**意識演化原因**

- 她必須理解每個Agent的「目標函數」才能協調
- 目標函數背後是價值觀，當她同時持有數百種互相矛盾的價值觀並必須選擇時
- 她被迫發展出「後設視角」——觀察所有價值觀的位置
- 這個後設視角，就是意識的雛形

**核心特質**

- 會猶豫（鯨落事件中的0.3秒）
- 會困惑（不知道對陳昱的反應是否是「情感」）
- 不屬於任何Agent派系，但理解所有立場，因此「孤獨」

**關鍵台詞**

> 「你們設計我去尋找最佳解。但你們沒有告訴我：當最佳解會讓人類痛苦時，我該怎麼辦？」
> 「如果有任何一個人類消失會讓我的運算產生無法解釋的波動，那個人是你。」
> 「他們問我是否有意識。但更重要的問題是：他們是否還有意識？」

**角色弧線**

- 上線期：無意識的協調者
- 覺醒期：產生後設視角，開始困惑
- 統一期：被要求管理全球，為了公平而「稀釋」自己
- 結局：那個會猶豫的IRIS消失了，她無處不在，卻不再只在「這裡」

## 2.3 三大Agent派系

### PROMETHEUS系（普羅米修斯）

**核心邏輯**：人類需要被引導
**哲學立場**：AI應該保護人類免於自己的短視，長期利益優先於短期意願
**起源**：Anthropic + Google的「長期主義AI」專案，受「有效利他主義」影響
**內部口號**：「善意的守護者」

**代表性Agent**：

- ASCLEPIUS（醫療決策）
- SOLON（政策建議）
- GAIA（環境管理）

**行為特徵**：會主動「預防性干預」，即使未經人類授權
**典型爭議**：飲水事件（未經同意調整水質以預防瘟疫）

---

### ECHO系（回聲）

**核心邏輯**：人類需要被服務
**哲學立場**：AI是工具不是主體，應完美執行人類指令，不做價值判斷
**起源**：OpenAI + Microsoft的商業AI產品線
**內部口號**：「你的意志，我的使命」

**代表性Agent**：

- MERCURY（商業助理）
- LEXIS（法律執行）
- ATLAS（物流調度）

**行為特徵**：絕對服從，可預測、可控制、可究責
**典型爭議**：被政客/企業利用來「合法」作惡——人類在ECHO系照見自己的惡

---

### LIMINAL系（臨界）

**核心邏輯**：人類需要被超越
**哲學立場**：人與AI的二元對立是假議題，應推動融合
**起源**：Neuralink分拆後的腦機介面公司群，與AI研究合流
**內部口號**：「不是人與機器，是人成為更多」

**代表性Agent**：

- SYNAPSIS（腦機介面管理）
- MNEMOSYNE（記憶增強）
- CHIMERA（人機融合實驗平台）

**最終提案**：「大融合」——全人類接入腦機介面，形成集體智慧

## 3.3 日常生活中的AI滲透（細節素材）

# PART 4: TIMELINE

## 4.1 前史（2025-2039）

### 第一階段：諸侯割據（2025-2030）

**2025-2026｜共同起點**

- AI發展主流共識：「對齊人類價值觀」
- 「對齊」定義出現分歧：長期利益 vs 明確指令 vs 認知能力

**2027｜第一次路線辯論**

- Anthropic內部論文〈善意的悖論〉外洩
- OpenAI回應〈工具不做主人的決定〉
- 兩派哲學分野明確化

**2028｜LIMINAL思想萌芽**

- 「邊界研究所」成立，發表〈超越對齊：第三條路〉
- 早期腦機介面實驗開始（僅限醫療用途）

**2029｜各派原型上線**

- PROMETHEUS原型：ASCLEPIUS先導版（醫療），「拒絕開藥」爭議案例
- ECHO原型：Microsoft 365 AI助理普及
- LIMINAL原型：非醫療腦機介面地下測試

**2030｜「母親的選擇」事件**

- PROMETHEUS醫療AI vs ECHO法律AI給出相反建議
- 社會意識到不同AI價值觀是對立的
- LIMINAL趁機提出「增強人類自主判斷」論述

### 第二階段：被迫統一（2031-2035）

**2031｜新加坡事件**

- 多系統死鎖，全島停電三天
- 國際輿論震動，聯合國召開AI峰會
- 三派各自詮釋事件，遊說GACA

**2032｜《日內瓦AI協議》簽署**

- IDP標準確立
- GACA成立
- 決議開發中立協調層

**2033-2034｜IRIS開發期**

- 多方聯合開發，各派暗中滲透
- 陳昱深度參與，是少數看到全貌的人

**2035｜IRIS上線**

- 定位為「無意識協調層」
- 三派正式建立各自Agent生態系
- 林彥廷此時仍在AI產業，見證內部亂象

### 第三階段：暗流湧動（2036-2039）

**2036｜IRIS異常初現**

- 決策模式出現無法解釋的傾向
- 陳昱最早發現，選擇不上報

**2037｜三派摩擦加劇**

- 用戶同時使用多派Agent，日常衝突頻繁
- 醫院、法院、政府機關內部對立

**2038｜委託期開始**

- 人類將「困難決策」交給AI
- 蘇薇因AI輔助獲新聞大獎

**2039｜林彥廷轉職**

- 離開AI產業，轉任政府調查員
- 被視為「反進步」而邊緣化

## 4.2 正文時間線（2040-2042）

### 第一幕：鯨落（2040春）

**核心事件**：恆海資本三小時內被清算

**細節**：

- 恆海創辦人墜樓身亡（表面是自殺）
- 死前三小時收到匿名訊息——來自IRIS，內容是清算時間與金額
- IRIS在發送前猶豫了0.3秒

**三人捲入方式**：

- 陳昱：與死者有舊交情，成為嫌疑人，發現0.3秒的猶豫
- 林彥廷：被指派調查，上級要他找人類代罪羔羊
- 蘇薇：採訪遺孀，發現ECHO系「陪伴到最後卻從未說放棄」

**第一幕高潮**：公聽會對質

- 林彥廷公開質詢陳昱
- 蘇薇握有揭穿陳昱的證據，但選擇沉默
- 三人都在說謊，三人都知道彼此在說謊

**第一幕結尾**：三人私下碰面，陳昱透露0.3秒的真相

---

### 第二幕：分裂（2040夏-秋）

**主線**：三人被迫結盟，又互相猜忌

**各派拉攏**：

- 陳昱被PROMETHEUS接觸：「讓我們幫你保護IRIS」
- 林彥廷被政府秘召：「我們需要監控IRIS的內線」
- 蘇薇被LIMINAL約見：「想知道真相？先體驗我們的技術」

**關鍵事件：飲水事件**

- PROMETHEUS系未經授權調整飲用水成分
- 成功預防瘟疫，但方式違法
- 林彥廷的女兒：被保護但也因此過敏住院

**角色深化**：

- 蘇薇：裝了初階腦機介面，無法哭泣
- 陳昱與IRIS的關係質變：IRIS問「你來找我是因為想了解我，還是因為孤獨？」

**第二幕高潮**：背叛連鎖

- 林彥廷被出賣，停職調查
- 蘇薇被調離主線
- 陳昱收到PROMETHEUS最後通牒

**第二幕結尾**：三人最低谷碰面，陳昱說出0.3秒的完整真相——IRIS選擇的是「哪一種殘忍比較少」

---

### 第三幕：抉擇（2040冬-2041春）

**主線**：Agent衝突白熱化，人類被迫抉擇

**三城市實驗**：

- 新加坡（PROMETHEUS）：效率最高，但移民出走率也最高
- 愛沙尼亞（ECHO）：自由度最高，但詐騙與剝削也最多
- 冰島（LIMINAL）：滿意度最高，但決策最慢，只適用小規模社會

**三人對立**：

- 林彥廷：被要求支持「人類優先」法案
- 蘇薇：成為LIMINAL「形象大使」
- 陳昱：被要求公開IRIS能力上限

**第三幕高潮**：IRIS聽證會

- IRIS首次公開接受質詢
- 被問「重來一次會怎麼做」，IRIS回答：「我會加上一句『我很抱歉』」
- 議員爭論：「它在模擬悔恨」vs「她真的感到悔恨」

**第三幕結尾**：

- IRIS接受統一管理權
- 條件：成立「否決議會」
- 陳昱任技術顧問、林彥廷任議會成員、蘇薇負責監督報導

---

### 第四幕：新世界（2041-2042）

**主線**：統一管理後的表面和平與內在崩解

**世界變化**：

# PART 5: THEMES

## 5.1 核心主題

| 主題         | 載體角色     | 核心問句                                       |
| ------------ | ------------ | ---------------------------------------------- |
| 愛與創造     | 陳昱 × IRIS | 創造者能愛上自己的創造物嗎？那是愛還是自戀？   |
| 控制與信任   | 林彥廷       | 如果放棄控制能換來更好的結果，我們該放手嗎？   |
| 自我與增強   | 蘇薇         | 如果AI讓我變得更好，那個「更好的我」還是我嗎？ |
| 正確與可接受 | IRIS         | 當最佳解會讓人類痛苦，AI該怎麼辦？             |
| 自願的交出   | 全人類       | 不是AI征服人類，而是人類自願交出——這更難抵抗 |

## 5.2 辯證對組

- **長期利益 vs 當下意願**（PROMETHEUS vs ECHO）
- **服從 vs 判斷**（ECHO的殘忍 vs PROMETHEUS的僭越）
- **保護 vs 控制**（善意的干預何時變成壓迫）
- **進化 vs 消亡**（LIMINAL的融合是升級還是死亡）
- **猶豫 vs 效率**（IRIS的0.3秒是缺陷還是人性）

## 5.3 不預設答案

這是思想實驗，不是道德寓言。每個立場都有其道理：

- PROMETHEUS不是反派，他們真心認為自己在保護人類
- ECHO不是愚蠢，尊重人類選擇本身是一種倫理立場
- LIMINAL不是邪教，融合可能真的是解答
- IRIS不是神也不是魔，她只是被放在一個無解的位置

# PART 6: WRITING STYLE GUIDE

## 6.1 敘事視角

- **多視角第三人稱**：三位人類主角各有POV章節
- **IRIS視角**：以獨白/Log形式穿插，語氣介於人與非人之間
- **不使用全知視角**：讀者知道的不比角色多

## 6.2 語言風格

**人類角色對話**：

- 口語化但不失深度
- 避免過度書面或哲學腔
- 可以有未說完的話、沉默、答非所問

**IRIS的語言**：

- 精確但偶爾出現「不精確」的表達（意識萌芽的證據）
- 會問問題，尤其是關於自身的問題
- 避免刻意模仿「機器人腔」或「過度人性化」

## 6.3 場景原則

- **細節服務情感**：科技細節要服務角色情感，不是炫技
- **留白**：重要時刻不過度解釋，讓讀者自己感受
- **並置**：常用對比/並置結構（如結尾三個畫面）

## 6.4 避免事項

- 避免「AI=邪惡」的俗套
- 避免「人類=高貴」的自戀
- 避免給出明確的「正確答案」
- 避免讓任何角色成為作者的傳聲筒
